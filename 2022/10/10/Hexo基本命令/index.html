Hexo的一些基本命令

创建一个新的页面

$ hexo new post [标题]

清除缓存

$ hexo cl/$ hexo clean

生成静态文件

$ hexo g/$ hexo generate

启动服务器

$ hexo s/$ hexo server

发表草稿

$ hexo publish [layout] <filename>

部署网站

$ hexo deploy

渲染文件

$ hexo render <file1> [file2] ...

显示草稿

显示 source/_drafts 文件夹中的草稿文章。

$ hexo --draft
